Bem-vindo ao Anchangelus!

Ha tres tipos de usuario: admin, medico e paciente.

O admin pode cadastrar medicos e pacientes, o medico pode cadastrar pacientes e o paciente pode cadastrar relatorios.

O sistema possui relatorios de diversos aparelhos, como medidor de pressao, que com auxilio de IOT, podem enviar uma requisicao POST para:

http://localhost:8080/WebService/rest/relatorios

Com corpo:

{
	"valor": 200.0,
	"notas": "Um copo de 200ml as 15:00",
	"tipo": "AGUA",
	"data": 1526083200000,
	"paciente": {
		"codigo": 3
	}
}

Assim atualizando automaticamente os dados do paciente, tambem possui um app no qual o usuario pode inserir os resultados manualmente.

Nao estao sendo usados acentos nos projetos devido aos problemas do Eclipse.

Por causa da materia ainda nao ter chegado em permissoes de login, nao ha bloqueios caso um usuario tente acessar algo que ele nao tenha permissao no webservice, criptografia de senhas ou algumas validações.
